# Recursos e funcionalidades

## Autenticação

- cadastro com e-mail e senha
- login com e-mail e senha
- login com Google
- verificação de e-mail
- recuperação de senha

## Perfil do usuário

- nome, e-mail e objetivo financeiro
- renda mensal
- avatar opcional
- atualização do perfil

## Dashboard

- visão geral do saldo e fluxo
- cards com contas e ativos
- lista de transações
- notificações contextuais
- alternância entre tema claro e escuro
- idioma português, inglês e espanhol

## Conexões bancárias

- integração com Pluggy Connect
- fluxo para conectar instituições financeiras
- indicação de status de sincronização

## Dados e sincronização

- fallback para dados mockados quando o backend não estiver disponível
- integração opcional com Data Connect para usuários e transações
- atualização de status de transações via UI

## Experiência do usuário

- layout responsivo
- tela de acesso amigável
- feedback visual para carregamento e erros
- compatibilidade com navegadores modernos
